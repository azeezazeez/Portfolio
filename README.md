## ✨ Key Features

- 🌙 **Light/Dark Theme Toggle** – Seamless user preference support  
- ⚡ **Smooth Scroll Animations** – Engaging scroll-triggered effects  
- 📱 **Fully Responsive** – Optimized for all devices (mobile, tablet, desktop)  
- 🎯 **Clean & Modern UI** – Minimalistic but powerful design  
- 🔗 **Social Media Integration** – Quick links to GitHub, LinkedIn, etc.  
- 📄 **Resume Actions** – View & download resume in one click  
- 💼 **Project Showcase** – Live demos and tech details for each project
