## 📌 About The Portfolio

This is a fully responsive personal portfolio website built to highlight my technical skills, featured projects, academic background, and contact information. Designed with a clean UI, smooth animations, and modern web practices for an exceptional user experience across all devices.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🌙 Theme Toggle | Switch between Light and Dark mode seamlessly |
| ⚡ Animations | Smooth scroll-triggered animations throughout |
| 📱 Responsive | Fully optimized for mobile, tablet, and desktop |
| 🎯 Clean UI | Modern, minimal, and professional design |
| 🔗 Social Links | Integrated social media profile links |
| 📄 Resume | View and download resume directly from the site |
